# Single page application using Laravel, JWT and Vuejs

----

## Stack
 * Mysql
 * Laravel
 * VueJs


## Usage
 * composer install
 * php artisan key:generate
 * php artisan jwt:secret
 * php artisan migrate:refresh
 * php artisan db:seed
